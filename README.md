# Telemetry Data Format Converter

This Python project converts two different telemetry JSON formats into a unified standard format.

## Features
- Converts Format 1 telemetry data to unified format
- Converts Format 2 telemetry data to unified format
- Converts ISO 8601 timestamp to milliseconds since epoch
- Includes unit tests to validate the conversion

## Files
- data-1.json → Telemetry format 1
- data-2.json → Telemetry format 2
- data-result.json → Expected unified format
- main.py → Conversion logic and unit tests
